# Панель инструментов

Панель расположена вертикально в левом верхнем углу и начинается с кнопок **Edit** и **Model**. В Edit доступны инструменты перемещения, вращения, масштабирования и рамки. В Model отображаются вершины мешей и появляется дополнительная панель выбора **Verts**, **Edges** или **Faces**. Соответственно режиму можно перетягивать точки, рёбра или весь полигон. Если выбрать две соседние вершины, активируется общее ребро. Нажмите **E** для Extrude, **F** чтобы создать грань из выбранного ребра и **L** для ``Loop Cut``. Кнопка **Fill** включает заливку, а контекстное меню содержит эти действия. Кнопка **Union** объединяет выбранные фигуры в один меш. Отдельная кнопка **Local** переводит гизмо в локальные координаты. В режиме перемещения между стрелками находится маленький квадратик, за который можно тянуть сразу по двум осям.
Кнопка *Screenshot* открывает диалог сохранения текущего вида в файл ``PNG``.

Правый клик по вьюпорту создаёт спрайты, пустышки, камеры или простые фигуры, включая полигоны.
Под меню располагается компактная панель с кнопками для быстрого включения или
выключения сетки, линейки, координат курсора и локального режима. Панели
инструментов закреплены и не показывают точек для перетаскивания.
