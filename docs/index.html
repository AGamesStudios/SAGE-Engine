Getting Started

  1. Install deps (Python + optional: LuaJIT/lupa, pythonnet, pybind11).
  2. Create your first plugin with tools/plugin_crafter.py.
  3. Run engine with selected plugins enabled.

Plugin Types

Shader / Image / Input / Script / Renderer / Camera / World ... compose them as you need.
