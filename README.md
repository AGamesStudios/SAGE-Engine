# SAGE Engine

Minimal starting point for the SAGE Engine.

## Commands

```bash
python -m pip install -e .
python examples/hello_sprite/main.py
pytest -q
```

## Папки проекта

- `src` - исходники движка
- `examples` - примеры
- `tests` - тесты
- `docs` - документация
- `tools` - служебные скрипты
