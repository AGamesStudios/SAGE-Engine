# SAGE-Engine
SAGE Engine — самый простой и легкий 3D-движок, предназначенный исключительно для создания игр. Его цель — стать самым простым и легким 3D-движком, с которым можно создать игру БЕЗ программирования и ограничений. Проект является полностью открытым (Open Source) и распространяется под лицензией MIT.
