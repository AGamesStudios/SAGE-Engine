# SAGE Engine

## Описание проекта

SAGE Engine — самый простой и легкий 3D-движок, разработанный исключительно для создания игр. Движок оптимизирован для 3D-игр и предоставляет все необходимые инструменты для создания игры **без сложного программирования**. Проект является полностью открытым (Open Source) и распространяется под лицензией MIT.

## Цели проекта

- **Простота использования:** Обеспечить интуитивно понятный и минималистичный подход к разработке игр, позволяющий создавать проекты даже начинающим разработчикам.
- **Легкость и производительность:** Создать максимально легкий и оптимизированный движок, ориентированный исключительно на игры.
- **Открытость и доступность:** Сделать движок доступным для всех, позволяя изучать, изменять и расширять его функциональность.
- **Без ограничений:** Разработать платформу, где можно создавать игры без жестких ограничений по функционалу или сложности.

## Основные особенности

- Простой и понятный интерфейс.
- Оптимизация под 3D-игры.
- Возможность расширения функционала без сложных переписываний.
- Полностью open source с лицензией MIT.
