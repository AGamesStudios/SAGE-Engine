Название проекта:
-----------------
SAGE Engine

Описание проекта:
-----------------
SAGE Engine — самый простой и легкий 3D-движок, предназначенный исключительно для создания игр.
Его цель — стать самым простым и легким 3D-движком, с которым можно создать игру БЕЗ программирования и ограничений.
Проект является полностью открытым (Open Source) и распространяется под лицензией MIT.

Особенности:
-----------
- Простота использования: Даже новички смогут быстро начать создавать игры.
- Оптимизирован для 3D-игр: Специально настроен для игр с трехмерной графикой.
- Легкая расширяемость: Архитектура движка позволяет легко добавлять новые функции и модули.
- Open Source: Полный исходный код доступен для всех, вы можете свободно изменять и дорабатывать его под свои нужды.

Установка и запуск:
--------------------
1. Клонирование репозитория:
   $ git clone https://github.com/your_username/SAGE-Engine.git
   $ cd SAGE-Engine

2. Установка зависимостей:
   Проект написан на Python и использует библиотеку Ursina.
   Убедитесь, что у вас установлен Python, затем выполните:
   $ pip install ursina

3. Запуск движка:
   Перейдите в каталог с исходным кодом:
   $ cd src
   Запустите движок:
   $ python sage_engine.py

Как работает SAGE Engine:
--------------------------
1. Инициализация:
   - При запуске движка создаётся экземпляр SAGEEngine.
   - Настраивается окно приложения, базовые параметры и загружаются скрипты из файла game_logic.ssc.

2. Система ролей (SAGERole):
   - Каждая роль представляет игровой объект: игрок, карта, камера, свет и т.д.
   - Роли можно легко расширять и настраивать через скрипты или наследование в коде.

3. Система ввода (SAGEInput):
   - Позволяет привязывать обработчики к клавишам.
   - При удержании клавиши, действие выполняется каждый кадр, что обеспечивает плавное управление.
   - Команды привязываются через скрипты и регистрируются в движке.

4. Физика:
   - Для ролей типа "player" реализована простая физика: гравитация и обработка прыжка.
   - Гравитация настраивается через скрипт, что позволяет регулировать поведение игрока.

5. Скрипты:
   - Файл game_logic.ssc содержит команды для создания ролей, настройки карты, управления окном и привязки ввода.
   - Команды парсятся и выполняются при инициализации движка.
